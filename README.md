# 卒業研究２ 実験装置

## 使用方法

1. このリポジトリをクローンまたはzip形式でダウンロード＆展開しておく。
```bash
git clone https://github.com/Suzuka-Y/B4-2-exp.git
```

2. Processing をインストールする（[リンク先](https://processing.org/)の "Download" をクリックし、指示に従う）。

3. クローンしたリポジトリの ".pde" の拡張子のついたファイルをダブルクリックで開く。

4. 現れたエディターの左上あたりにある実行ボタン（三角マーク）を押して、実験を進める。
